# Lingo
# {EPITECH} | First Years

"Lingo game : One player have multiple tries to
guess a Word
In France it's called Motus."

===============================================

Exemple :

**_**: invalid letter \
**?**: incorrectly placed letter \
Will you find the secret word ? \
p _ _ _ _

Round 1 \
**>petit** \
p _ _ _ _ \

Round 2 \
**>panda** \
panda \
You won!
